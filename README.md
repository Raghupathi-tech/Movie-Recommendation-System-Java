# Movie Recommendation System

## Description
A simple recommendation system developed in Java that suggests movies based on user-selected genres.

## Features
- Genre selection
- Movie recommendations
- Interactive console interface
- Decision-making logic

## Technologies Used
- Java
- Switch Case Logic

## How to Run
1. Open the project in Eclipse.
2. Run MovieRecommendation.java.
3. Select a genre to get recommendations.
